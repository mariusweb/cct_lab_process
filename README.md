# Getting Started 

Open terminal and type in

### `git clone git@github.com:mariusweb/cct_lab_process.git .`
or
### `git clone https://github.com/mariusweb/cct_lab_process.git .`

then type in 

### `npm install`

and then 

### `npm start`

Your browser should open [http://localhost:3000](http://localhost:3000)
